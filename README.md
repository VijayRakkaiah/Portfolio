## How to use / deploy to GitHub Pages

1. Create a new repository on GitHub (example name: `vijay-rakkaiah-portfolio`).
2. Copy the folder contents into the repo (push to `main` branch).
3. Make sure the folder structure is preserved:
   - `index.html` at root
   - `assets/` folder with `css/`, `js/`, `img/`, `docs/`
4. On GitHub, go to **Settings > Pages** (or **Site settings > Pages**), choose branch `main` and folder `/ (root)` then Save.
5. After a minute, your site will be live at `https://<your-username>.github.io/<repo-name>/` or for a user/organization repo `https://<your-username>.github.io/`.
